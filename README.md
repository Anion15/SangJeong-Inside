[상정인사이드](https://commu.world)는 회원가입 없이 바로 익명으로 사용할 수 있는 자유로운 토론 커뮤니티입니다. 고정되지 않는 익명성에 초점을 맞췄습니다.
작성자의 식별 정보 대신, 매시간 바뀌는 익명 ID를 기반으로 글과 댓글이 기록됩니다. 이를 통해 누구인지 특정되기보다, 오직 생각과 의견 그 자체에 집중할 수 있는 환경을 만드는 것을 목표로 합니다.

상정인사이드는 단순한 익명 게시판에 머무르지 않고, 내 활동 확인, 검열 기능, 정보 실시간 반영, 이미지 및 링크 파싱, 댓글 접기 등 사용자가 더 편하고 안전하게 커뮤니티를 이용할 수 있도록 다양한 기능을 함께 제공합니다.
